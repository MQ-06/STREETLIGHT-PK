# Municipal & Admin Web Interface

Built with Vite + React + Tailwind CSS + React Router.

## Setup
```bash
npm install
npm run dev
```

## Structure
- `src/pages/` — Add new pages here
- `src/components/` — Shared reusable components
- `src/layouts/` — Layout wrappers
- `src/routes/router.jsx` — All route definitions
- `src/hooks/` — Custom React hooks
- `src/utils/` — Helper/utility functions
- `src/assets/` — Images, icons, logos
